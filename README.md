# Simple QR Maker

Simple QR Maker is a client-side custom QR code generator that runs in the browser. It lets you style QR codes, add an optional center image, and export square PNG files.

The interface uses a monochrome, utilitarian style with restrained surfaces, clear controls, and a focused tool workspace.

## Features

- Custom QR data for URLs or text
- Body, eye frame, and eye ball shape choices
- QR and background color controls
- Optional transparent background
- Center image upload for logos or icons
- Logo size control
- Error correction level selection
- PNG export from 150 x 150 px to 1000 x 1000 px

## Usage

1. Open `index.html` in a browser.
2. Enter the URL or text for the QR code.
3. Customize the shapes, colors, background, and error correction.
4. Upload an optional center image and adjust its size.
5. Choose the PNG output size with the export slider.
6. Click **Download PNG**.

All QR code generation happens in your browser. No server is required.
